# Project

Small app to retrieve jokes about Chuck Norris, categorize and list.

Developed with Vue.js and Vuex.


### Project setup

```
yarn install
```

#### Compiles and hot-reloads for development

```
yarn serve
```

#### Compiles and minifies for production

```
yarn build
```

#### Run your unit tests

```
yarn test:unit
```

#### Lints and fixes files

```
yarn lint
```
